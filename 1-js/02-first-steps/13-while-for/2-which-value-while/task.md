# Ce valori afișează bucla while?

Pentru fiecare iterație a buclei, scrie ce valori afișează și apoi compară-le cu soluția.

Ambele bucle afișează prin funcția `alert` aceleași valori, sau nu?

1. Forma cu prefix `++i`:

   ```js
   let i = 0;
   while (++i < 5) alert( i );
   ```
2. Forma cu sufix `i++`

   ```js
   let i = 0;
   while (i++ < 5) alert( i );
   ```
