# Rescrieți instrucțiunea "switch" folosind "if"

Folosind `if..else`, scrieți codul care ar corespunde următorului `switch`:

```js
switch (browser) {
  case 'Edge':
    alert( "Folosiți Edge!" );
    break;

  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    alert( 'OK, suportăm și aceste browsere' );
    break;

  default:
    alert( 'Sperăm că această pagină arată bine!' );
}
```
